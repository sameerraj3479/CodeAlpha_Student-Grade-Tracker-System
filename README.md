# Student Grade Tracker System

A Java-Based Student Performance Analysis System Built Using Object-Oriented Programming and ArrayList

Student Records • Grade Analysis • Performance Tracking • Report Generation

## Project Overview

Student Grade Management System is a console-based Java application designed to manage student academic records and perform basic performance analysis.

The system allows users to enter multiple student names and grades, store them efficiently using Java Collections, and generate a detailed summary report. It automatically calculates the average score, identifies the highest-performing student score, and determines the lowest score among all students.

This project was developed to strengthen understanding of Java programming fundamentals, Object-Oriented Programming principles, collection frameworks, user input handling, and data processing techniques.
